Information Technology Projects

Active Measurements of Routes, Losses, and Delays (tracert)

In this paper I analyze the connection between my local machine to the network locations of five different websites across the globe. I trace the route from my machine to the remote locations as well as measure packet loss and round-trip time ten times over the span of a week. I trace the route and measure packet loss and round-trip time for the following five websites: www.microsoft.com, www.ford.com, www.dhs.gov, www.ox.as.uk, www.momat.go.jp.

Head to Head Database Web Application

In this project, I make a web application to access Head to Head match data from a database. The web application lets you search a player's name in a searchable dropdown and then displays the players' head to head record with another player, what tournament(s) they played at, the matchup winner and loser at that tournament, and the date of the tournament. I'm not hosting this database web application anywhere but there are screenshots in the document if you are interested.

Calculator Web Service
